### 必备物品

* 防晒霜、充电宝、离线地图
* 超市: 洗漱用品、毛巾、拖鞋、保鲜袋、小刀、纸巾、矿泉水、食品(干粮、水果)
* 单反、小型望远镜、药(感冒冲剂、坏肚子药、板蓝根)

* 驾驶证、身份证

* 拍照注意当地人所要金钱
